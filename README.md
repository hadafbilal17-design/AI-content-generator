# AI Travel Blog Generator

An AI-powered web app that creates travel blogs instantly based on destination and preferences.

## Features
- Generate travel blogs in seconds
- User-friendly interface
- Custom destination input
- Powered by AI models

## Tech Stack
- Python
- Streamlit
- Hugging Face Models

## How to Run
1. Install requirements:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py

## Purpose
This project was built to help users create travel content quickly and creatively.
